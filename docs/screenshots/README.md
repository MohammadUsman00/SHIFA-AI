# Screenshots

PNG files in this folder are **representative UI mockups** in the Shifa AI visual style (dark theme, teal accents, glass cards).

For **pixel-accurate** marketing or store listings, capture your own from a running build:

1. `npm run dev` → open http://localhost:3000  
2. Use the browser devtools device toolbar or full window at 1440×900.  
3. Export PNG/WebP and replace the files here (keep the same filenames if you want README links to stay valid).
